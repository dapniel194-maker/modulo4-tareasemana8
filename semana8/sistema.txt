El comando uname -a muestra información del sistema operativo, como el nombre del sistema, la versión del kernel, la arquitectura del procesador y otros datos técnicos. El comando whoami muestra el nombre del usuario que está utilizando la sesión actual. Conocer esta información es útil al administrar un servidor porque permite identificar el sistema en uso, verificar el usuario con el que se trabaja y facilitar tareas de administración, mantenimiento y seguridad.
